# 🚀 Backend Data Pipeline

## 📌 Overview
This project implements a complete backend data pipeline using:

- Flask (Mock API Server)
- FastAPI (Data Ingestion Service)
- PostgreSQL (Database)
- Docker Compose (Container orchestration)

The pipeline flow is:
Flask API → FastAPI Ingestion → PostgreSQL → API Response

---

## ⚙️ Tech Stack
- Python 3.10+
- Flask
- FastAPI
- SQLAlchemy
- PostgreSQL
- Docker & Docker Compose
